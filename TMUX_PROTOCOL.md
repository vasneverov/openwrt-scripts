# Tmux протокол параллельного ремонта

## Запуск

```bash
cd ~/CLAUDECODE
chmod +x tmux-router-repair.sh
cd ~/CLAUDECODE && ./tmux-router-repair.sh
```

## Структура окна

```
┌─────────────────┬──────────────┐
│                 │   АГЕНТ 1    │
│   КООРДИНАТОР  │   (панель 1) │
│   (панель 0)   ├──────────────┤
│                 │   АГЕНТ 2    │
│                 │   (панель 2) │
└─────────────────┴──────────────┘
```

## Управление

### Переключение между панелями
- `Ctrl-b` затем стрелка (→ ← ↑ ↓) — перемещение
- `Ctrl-b q` — показать номера панелей, затем цифра — перейти
- `Ctrl-b o` — следующая панель по кругу

### Обновление статуса агента

В панели агента запускаешь:
```bash
~/CLAUDECODE/tmux-status-updater.sh [pane] "[статус]" "[IP]" "[задача]"
```

Примеры:
```bash
# Агент 1 начал работу
~/CLAUDECODE/tmux-status-updater.sh 1 "🔄 РАБОТАЕТ" "100.122.120.65" "Ремонт S78-01"

# Агент 1 готов
~/CLAUDECODE/tmux-status-updater.sh 1 "✅ ГОТОВО" "100.122.120.65" "Ключи заменены"

# Агент 2 ошибка
~/CLAUDECODE/tmux-status-updater.sh 2 "❌ ОШИБКА" "100.90.240.1" "Tailscale offline"
```

## Протокол работы

### 1. Назначение (Координатор)
```
ПАНЕЛЬ 0 (Координатор):
├── Отдаю задачи агентам
├── Проверяю результаты
├── Обновляю общий статус
└── Синхронизирую действия

ПАНЕЛЬ 1 (Агент 1): IP [указать] | Задача [указать]
├── Диагностика
├── Ремонт
└── Отчёт

ПАНЕЛЬ 2 (Агент 2): IP [указать] | Задача [указать]
├── Диагностика
├── Ремонт
└── Отчёт
```

### 2. Тайминги

| Этап | Время | Контрольная точка |
|------|-------|-------------------|
| Диагностика | 0-2 мин | Статус: 🔍 ДИАГНОСТИКА |
| Создание ключей | 2-4 мин | Статус: 🔧 КЛЮЧИ |
| Установка | 4-6 мин | Статус: ⚙️ УСТАНОВКА |
| Проверка | 6-8 мин | Статус: ✅ ГОТОВО |

## Горячие клавиши tmux

| Команда | Действие |
|---------|----------|
| `Ctrl-b %` | Разделить вертикально |
| `Ctrl-b "` | Разделить горизонтально |
| `Ctrl-b x` | Закрыть текущую панель |
| `Ctrl-b z` | Развернуть панель на весь экран (toggle) |
| `Ctrl-b [` | Режим копирования/прокрутки |
| `Ctrl-b d` | Отключиться от сессии (не закрывая) |
| `tmux attach` | Подключиться обратно |

## Синхронный ввод (опасно!)

Чтобы печатать одновременно во все панели:
```bash
Ctrl-b :setw synchronize-panes on
```

Выключить:
```bash
Ctrl-b :setw synchronize-panes off
```

⚠️ **Внимание:** При синхроне команды идут ВО ВСЕ панели одновременно!
