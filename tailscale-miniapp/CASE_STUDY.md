## 🎯 С чего всё началось

Василий управляет сетью из **283 роутеров** по всей России.
Каждый роутер подключён через **Tailscale** — VPN-сервис, который даёт каждому устройству постоянный IP.

Роутеры распределены по трём бесплатным учётным записям Tailscale
(бесплатный план — до 100 устройств на аккаунт).

**Проблема:** Чтобы проверить статус любого роутера, нужно было:
1. Открыть браузер
2. Зайти в Tailscale
3. Найти нужное устройство
4. Скопировать IP
5. Открыть панель роутера по этому IP

Это занимало минуты, а нужна была **секунда прямо из телефона**.

## 🔍 Что сделал Claude перед тем как писать код

**1. Собрал реальные данные**

Запросил все три Tailscale API и получил полный список устройств:
```
GET https://api.tailscale.com/api/v2/tailnet/{email}/devices?fields=all
```

Выяснил нюанс: поле `online` в API не существует —
статус нужно брать из `connectedToControl`. Это типичная ловушка,
которую разработчик нашёл бы через час отладки.

**2. Отфильтровал лишнее**

В учётках были не только роутеры, но и маки и телефоны.
Написал фильтр по типу ОС — в список попадают только роутеры.

**3. Определил архитектуру**

- **Бэкенд:** FastAPI (Python) — запрашивает Tailscale API, кэширует 30 сек
- **Фронтенд:** чистый HTML/JS — работает как Telegram Mini App
- **Бот:** python-telegram-bot — одна кнопка, открывает Mini App
- **Деплой:** systemd на существующем VPS, nginx + Let's Encrypt

## ⚙️ Что было написано и сделано

### Бэкенд (`app.py`)
- Подключается к трём Tailscale-учёткам параллельно
- Кэш на 30 секунд — чтобы не долбить API при каждом открытии
- Форматирует дату последнего онлайна: `2026-04-07T08:55:10Z` → `07 апр 08:55`
- Два эндпоинта: `/api/devices` (кэш) и `/api/refresh` (принудительное обновление)

### Telegram-бот (`bot.py`)
- Только для одного пользователя (ADMIN_ID) — приватный инструмент
- Команда `/start` → кнопка «📡 Открыть монитор»
- Команда `/status` → быстрый текстовый ответ: всего/онлайн/офлайн

### Mini App (`index.html`)
Полноценное мобильное приложение внутри Telegram:

| Функция | Как работает |
|---|---|
| **Живой поиск** | Фильтрует по имени или IP прямо при вводе |
| **Фильтр статуса** | Все / 🟢 Онлайн / ⚫ Офлайн |
| **Фильтр учёток** | Все / Учётка 1 / 2 / 3 с счётчиком устройств |
| **Умный тап** | Онлайн → сразу браузер. Офлайн → детали |
| **Детали устройства** | IP (тап = скопировать), учётка, последний онлайн, дата добавления |
| **Авто-обновление** | Каждые 60 секунд без перезагрузки |
| **Счётчик лимита** | Под каждой учёткой: `99 / 100`, красным если перебор |

### Деплой на сервер
- nginx настроен как reverse proxy
- SSL-сертификат Let's Encrypt (HTTPS обязателен для Telegram Mini App)
- Два systemd-сервиса: для FastAPI и для бота
- Оба запускаются автоматически при перезагрузке сервера
