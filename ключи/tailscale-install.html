Tailscale — установка на WR3000H / OpenWrt 25.12

Cudy WR3000H · OpenWrt 25.12 · apk · gunanovo repo · serve → auth
1 Установка + запуск + serve + авторизация
wget -O /etc/apk/keys/gunanovo@github.io.pub https://gunanovo.github.io/openwrt-tailscale/key-build.rsa.pub && echo "https://gunanovo.github.io/openwrt-tailscale/$(cat /etc/apk/arch)/packages.adb" >> /etc/apk/repositories.d/customfeeds.list && apk update && apk add tailscale iptables ip6tables && killall tailscaled 2>/dev/null; sleep 2; uci set tailscale.settings.fw_mode=nftables; uci commit tailscale; /etc/init.d/tailscale enable; tailscaled --state=/var/lib/tailscale/tailscaled.state --socket=/var/run/tailscale/tailscaled.sock & sleep 3; tailscale serve --bg --tcp 80 tcp://localhost:80; tailscale serve --bg --tcp 443 tcp://localhost:443; tailscale serve --bg --tcp 22 tcp://localhost:22; tailscale up --accept-dns=false --accept-routes --reset; tailscale serve status
После вставки дождись ссылки login.tailscale.com/a/... — открой в браузере.
После авторизации увидишь Success и список портов.
2 Автозапуск serve после перезагрузки (rc.local)
echo -e '#!/bin/sh\n(sleep 10\ntailscale serve --bg --tcp 80 tcp://localhost:80\ntailscale serve --bg --tcp 22 tcp://localhost:22\ntailscale serve --bg --tcp 443 tcp://localhost:443) &\nexit 0' > /etc/rc.local && chmod +x /etc/rc.local && echo OK
Выполни после успешного шага 1. Должно вывести: OK
Если tailscale уже установлен — только запуск
killall tailscaled 2>/dev/null; sleep 2; tailscaled --state=/var/lib/tailscale/tailscaled.state --socket=/var/run/tailscale/tailscaled.sock & sleep 3; tailscale serve --bg --tcp 80 tcp://localhost:80; tailscale serve --bg --tcp 443 tcp://localhost:443; tailscale serve --bg --tcp 22 tcp://localhost:22; tailscale up --accept-dns=false --accept-routes --reset; tailscale serve status
