## Обнаруженные проблемы

### 1. Podkop не был запущен
```
/etc/init.d/podkop status → not running
```
**Исправление:** Запустил вручную.

### 2. Отсутствовал watchdog Podkop
Был только Tailscale watchdog, не хватало podkop-watchdog.

**Исправление:**
```bash
cat > /etc/podkop-watchdog.sh << 'EOF'
#!/bin/sh
if ! ps | grep -q "sing-box run"; then
    logger -t podkop-watchdog "sing-box not running, restarting podkop"
    /etc/init.d/podkop restart
fi
EOF
chmod +x /etc/podkop-watchdog.sh
echo "*/5 * * * * /etc/podkop-watchdog.sh" | crontab -
```

### 3. init.d/tailscale был ENABLED
Должен быть DISABLED для userspace-networking.

**Исправление:** `/etc/init.d/tailscale disable`

### 4. exclude_ntp = 0
Исправлено на `uci set podkop.settings.exclude_ntp='1'`

### 5. fw_mode был 'nftables'
Исправлено на `uci set tailscale.settings.fw_mode='none'`
