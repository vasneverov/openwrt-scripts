# Урок: Лимиты — всегда, автоматом

**Дата:** 2026-05-11

## Правило
При любом входе на роутер — **всегда добавлять лимиты**, если их нет.

## Что добавлять
Только одно:
```bash
# ulimit в rc.local (первой строкой)
sed -i '1i ulimit -n 65535' /etc/rc.local
```

**Больше ничего.** Никаких sysctl, никаких somaxconn, rmem, wmem.

## Почему
- ulimit по умолчанию 1024 — мало для sing-box
- 65535 — достаточно

- flash_router_universal.md

- rescue_generic.sh
- Все скиллы лечения

Не спрашивать. Не забывать. Добавлять всегда. Только ulimit -n 65535.
