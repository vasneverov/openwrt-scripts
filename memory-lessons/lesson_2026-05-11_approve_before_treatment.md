# Урок: Спрашивать одобрение перед лечением

**Дата:** 2026-05-11

## Правило
После диагностики и плана лечения — **обязательно спросить одобрение** у пользователя. Не начинать лечение без подтверждения.

## Алгоритм
1. Диагностика → отчёт
2. План лечения (что буду делать)
3. **Спросить: "Начинать?"**
4. Только после ✅ — выполнять

## Почему
- Пользователь хочет контролировать процесс
- Может быть несколько вариантов лечения
- Может быть другой приоритет

## Напоминание
Не лечить молча. Сначала план → одобрение → лечение.
