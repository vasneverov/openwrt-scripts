# Урок: Протокол полного отчёта после rescue + диагностики

## Когда применять
После того как:
1. Пользователь дал IP роутера
2. Сказал "закидывай спасительные скрипты" (rescue_generic.sh)
3. Нужно сделать полную диагностику

## Последовательность
1. Закинуть rescue_generic.sh на роутер
2. Запустить скилл **router-full-report** — создать `/tmp/diag_ROUTER_NAME.sh` и выполнить через SSH pipe
3. Сформировать отчёт по шаблону ниже
4. Вызвать `attempt_completion` с полным отчётом внутри `result` параметра
